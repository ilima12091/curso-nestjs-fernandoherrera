# Teslo API

1. Instalar las dependencias

```
yarn install
```

2. Clonar el archivo `.env.template` y renombrarlo a `.env` configurando las variables de entorno.
3. Levantar la base de datos

```
docker-compose up -d
```

4. Ejecutar la aplicaci√≥n en modo desarrollo

```
yarn start:dev
```
