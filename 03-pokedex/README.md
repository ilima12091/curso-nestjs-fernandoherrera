# Desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar la BD

```
docker-compose up -d
```

5. Reconstruir la BD con el seeder

```
http://localhost:3000/api/seed
```

## Stack

- NestJS
- MongoDB
